# Comandos importantes

## Kali Linux

**ping -c 3 "-n° do IP-"**  

*ping = envio de pacote ai IP informado*\\
*3 = Números de pacotes a ser enviado*\\
Usado para testar conexão com um IP\\
\\
**nmap -sV -p 21,22,80,445,139 "n° de IP"**\\
*sV = Tentativa de descobrir a versão que está o serviço/porta\\
Usado para descobrir quais serviços/portas estão abertas. Os números antes do IP são portas bastantes comuns a serem atacadas.\\
